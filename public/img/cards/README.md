### 🎨 Playing Cards Assets
This directory contains the playing card images used in the game.

### 📂 Folder Structure
The playing cards are stored in the following directory:

```
public/img/cards/ 

    ├── card-cover.svg
    ├── clubs_01.svg
    ├── ...    
    ├── clubs_13.svg
    ├── diamonds_01.svg
    ├── ...
    ├── diamonds_13.svg
    ├── hearts_01.svg
    ├── ...
    ├── hearts_13.svg
    ├── spades_01.svg
    ├── ...
    └── spades_13.svg
```
